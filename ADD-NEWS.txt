1. In _posts, create a file with the exact format of "%Y-%M-%D-title.markdown"
2. In that file, copy the header block (From the first line with 3 dashes to the last line) from past posts
3. Use markdown to write your post, here's some resources below:
https://github.com/adam-p/markdown-here/wiki/Markdown-Cheatsheet
https://jekyllrb.com/docs/posts/
4. In Command Prompt, run GenerateJekyll.sh
